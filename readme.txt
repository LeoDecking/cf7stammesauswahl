Auswahl von DPSG-Diözesanverband, Bezirk und Stamm in Contact Form 7 Formularen

== Description ==

Dieses Plugin ermöglicht es, bei Contact Form 7 Formularen seinen Stamm, Bezirk und Diözesanverband auszuwählen.
Dies vereinheitlicht die Stammes-Namen z.B. in Anmeldelisten.

Die aktuellen Gruppierungen werden einmalig (und bei Aktualisierungen natürlich) aus der Gruppierungsliste des Anhangs der aktuellen DPSG-Satzung geladen.

Das Plugin fügt einen neuen Tag für Contact Form 7 hinzu: `[stamm]`.
Es können Werte voreingestellt werden, sodass z.B. auf der Internetseite des Diözesanverbands Paderborn automatisch Paderborn als Diözesanverband ausgewählt ist.

Sollte sich der Stamm einer Person noch nicht in der Gruppierungsliste befinden, kann man seinen Stamm auch manuell eingeben.

**Einrichtung:** Nachdem das Plugin installiert wurde, wird in den Wordpress-Einstellungen eine Seite "Stammesauswahl" hinzugefügt.
Dort musst du einmalig die aktuelle Gruppierungsliste laden.


*Entwickelt von Leo Decking im FSJ im DPSG-Diözesanbüro Paderborn*
